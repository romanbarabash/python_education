Task is the main entity in this application. Everything revolves around tasks and managing them.
id (optional)
UUID A unique identifier for the task. These are created in ascending order. format: uuid

String The title for a task.
description (optional)
String The task description is a longer, more detailed description of the issue.

Integer format: int32
